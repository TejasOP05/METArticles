## Project Overview

MetArticles is a Flask web application for submitting, reviewing, and managing academic/research articles. It supports a role-based workflow with authors, supervisors, and administrators.

## Key Components

### 1. **Technology Stack**
- Flask — web framework
- SQLAlchemy — database ORM
- Flask-Login — authentication
- SQLite — database (with PostgreSQL/Neon support)
- Werkzeug — password hashing and utilities

### 2. **User Roles**
- Author: submit articles
- Supervisor: review and approve/reject
- Admin: full access, including user management

### 3. **Core Features** (from models and templates)

**User Management:**
- Registration and login
- Password change
- Role-based access control
- User management dashboard (admin)

**Article Management:**
- Submit articles with PDF uploads (max 16MB)
- Articles include: title, abstract, keywords, category
- Status tracking: pending, approved, rejected
- Download tracking
- Comments/reviews on articles

**Workflow:**
- Authors submit articles
- Supervisors review and approve/reject
- Tracking of submission and review timestamps
- Approval dashboard for supervisors
